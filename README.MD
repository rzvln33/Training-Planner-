## Описание программы

Training Planner — десктопное GUI‑приложение на Python для планирования и учёта спортивных тренировок. Программа помогает вести журнал тренировок, фильтровать записи по типу и дате, а также сохранять данные в формате JSON для долгосрочного хранения.

### Возможности

Программа позволяет:
* добавлять новые тренировки через удобную форму ввода;
* отображать все тренировки в виде интерактивной таблицы с сортировкой;
* фильтровать тренировки по типу (кардио, силовая, йога и т. д.) и по диапазону дат;
* сохранять все данные в локальный JSON‑файл (`trainings.json`) при закрытии программы;
* загружать сохранённые данные из JSON‑файла при запуске;
* проверять корректность вводимых данных:
    * формат даты (должен быть ГГГГ‑ММ‑ДД);
    * положительность значения длительности тренировки (в минутах);
* сбрасывать фильтры и возвращаться к полному списку тренировок;
* выводить понятные сообщения об ошибках при некорректном вводе данных.

## Инструкция по запуску

### Требования для запуска программы

Для корректной работы приложения необходимо:

1. **Установить Python версии 3.6 или выше**. Проверить версию можно командой:
   ```bash
   python --version
Примеры использования программы
Пример 1. Добавление новой тренировки
В главном окне программы найдите форму ввода данных с полями:

Дата (поле ввода);

Тип тренировки (выпадающий список или поле ввода);

Длительность (поле ввода, в минутах).

Заполните поля:

В поле Дата введите дату в формате ГГГГ‑ММ‑ДД, например 2026-05-20.

В поле Тип тренировки выберите из списка или введите вручную, например «Кардио».

В поле Длительность укажите количество минут, например 45.

Нажмите кнопку «Добавить тренировку».

Проверьте результат:

новая запись должна появиться в таблице ниже формы;

поля формы автоматически очистятся для следующего ввода;

если данные некорректны, появится сообщение об ошибке.

Пример 2. Фильтрация тренировок по типу
В блоке фильтров найдите выпадающий список «Тип тренировки».

Выберите нужный тип, например «Силовая».

Нажмите кнопку «Применить фильтр».

Наблюдайте результат:

таблица обновится — в ней останутся только тренировки выбранного типа;

остальные записи временно скроются.

Чтобы сбросить фильтр и вернуть полный список, нажмите кнопку «Сбросить фильтры».

Пример 3. Фильтрация по диапазону дат
В блоке фильтров найдите два поля:

«От» — начальная дата диапазона;

«До» — конечная дата диапазона.

Введите даты в формате ГГГГ‑ММ‑ДД:

в поле «От»: 2026-05-01;

в поле «До»: 2026-05-31.

Нажмите «Применить фильтр».

Проверьте таблицу:

в ней останутся только тренировки, проведённые в указанном месяце;

записи за другие даты скроются.

Пример 4. Сохранение и загрузка данных
Сохранение данных

Добавьте несколько тренировок через форму ввода (см. Пример 1).

Закройте программу одним из способов:

нажмите крестик (×) в углу окна;

выберите в меню «Файл» → «Выход» (если реализовано).

Программа автоматически сохранит все данные в файл trainings.json в директории проекта.
